Understanding Descriptive Text : Definition and Purpose

What is Descriptive Text and its purpose

Descriptive text is a text that describes a particular object in detail. The purpose of descriptive text is to describe an object in detail. The structure of descriptive text consists of two parts, namely identification and description. When writing descriptive text, there are linguistic rules that you must use. The language features of descriptive text consist of three, namely adjective, Simple Present Tense, and adverb of frequency.

The purpose of descriptive text is to describe an object in detail. The purpose of descriptive text is to describe or describe an object in detail (detailed). These objects can be people, animals, things, or places.

A. Structure of Descriptive Text

  1. The identification section contains the introduction of the object to be described. For example, 'what is an object?' or 'where did that object come from?'.
  2. The description section contains a more specific description of the object being described. In this section, you can explain the appearance and personality of the object.

B. Language Features of Descriptive Text

  1. Adjective Descriptive text is inseparable from adjectives or adjectives. When describing an object, we must use adjectives, such as big, small, tall, short, smart, friendly, and so on.
    Example: Hanni is so pretty. She has an oval face, rounded eyes, pointed nose, and long hair. She is smart and friendly.
    2. The adjectives used in the descriptive text above are : pretty (cantik), oval (oval/lonjong), rounded (bulat), pointed (mancung), long (panjang), smart (pintar), dan friendly (ramah).
  2. Simple Present Tense Simple Present Tense is commonly used to state facts or general truths, as well as customs. When describing an object, we will use this type of tense because what we describe from the object is fact.
  3. Adverb of Frequency The adverb of frequency is an adverb that is useful for indicating how often an activity is performed. Well, in descriptive text, we can explain the object's habit of using this.
    Example: Hevi is a helpful and polite person. She often helps me to do my homework and always respects the elders. From this description, the author uses adverbs of frequency, namely often and always.
